# Task 2 — Unemployment Analysis with Python

An exploratory analysis and interactive dashboard examining unemployment across
Indian states, areas, and time, with special attention to the COVID-19 period.

## Features

- Cleans blank rows and standardises dates and column names
- Tracks national unemployment over time
- Compares rural and urban areas
- Ranks states and zones
- Highlights the first COVID-19 lockdown period
- Exports cleaned datasets, summary tables, and publication-ready charts

## Run

```bash
python analysis.py
streamlit run app.py
```

## Dataset

The project uses `Unemployment in India.csv` and
`Unemployment_Rate_upto_11_2020.csv`, supplied with the internship task.
